# 광고뷰가 화면에 보이지 않는 경우 광고 요청 및 중단

CaulyAdView는 화면 표시 여부와 무관하게 광고 요청이 수행될 수 있습니다.\
따라서 화면 전환/탭 이동/리스트 재활용 등으로 **더 이상 광고 요청이 필요하지 않은 시점**에는 반드시 요청을 중단해야 합니다.

#### 광고 요청 중단 API

아래 명령을 호출하여 광고 요청을 중지할 수 있습니다.

```objc
[caulyAdView stopAdRequest];
```

**권장 호출 시점 예시**

* 화면이 사라지는 시점(`viewWillDisappear`, `deinit`, 셀 재사용 준비 등)
* 광고 노출을 더 이상 하지 않는 UI 상태 전환 시점
* 주기적 요청(리프레시)을 자체 구현했다면 중단 조건 발생 시점

{% hint style="warning" %}
광고뷰가 보이지 않더라도 네트워크 요청이 진행될 수 있으므로,&#x20;

**광고 영역이 더 이상 유효하지 않은 경우 stopAdRequest 호출을 권장**합니다.
{% endhint %}

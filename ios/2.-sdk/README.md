# 2. SDK 설치

{% hint style="info" %}
Cauly iOS SDK는 다음과 같은 설치 방식을 제공하며,

신규 연동 및 일반적인 프로젝트 환경에서는 **Swift Package Manager 또는 CocoaPods 사용을 권장합니다.**

* Swift Package Manager (권장)
* CocoaPods (권장)
* 수동 다운로드 (선택)
{% endhint %}

## 최신 SDK Version : 3.1.22

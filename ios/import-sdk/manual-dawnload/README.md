# 수동 다운로드

본 페이지에서는 **패키지 매니저를 사용할 수 없는 환경**에서\
Cauly iOS SDK를 연동하기 위한 **수동 다운로드 방식**에 대해 안내합니다.

***

### 수동 다운로드 방식이 필요한 경우

일반적인 프로젝트 환경에서는\
**Swift Package Manager 또는 CocoaPods 사용을 권장**합니다.

다만, 아래와 같은 경우에는 수동 다운로드 방식을 사용할 수 있습니다.

* 사내 보안 정책으로 외부 패키지 저장소 접근이 제한된 경우
* 내부 빌드 시스템에서 로컬 SDK 파일만 허용되는 경우
* 특정 SDK 버전을 고정하여 관리해야 하는 경우
* 패키지 매니저 사용이 불가능한 특수 환경

> 수동 다운로드 방식은 **권장 설치 방식이 아닌 대체 수단**입니다.

***

### 수동 다운로드 방식 개요

{% embed url="https://github.com/cauly/iOS-SDK/tree/master/Cauly3.1/CaulyLib" %}

수동 다운로드 패키지에는\
**두 가지 설치 방식이 함께 포함**되어 있습니다.

* **XCFramework 방식**
* **Static Library 방식**

프로젝트 환경에 따라 **두 방식 중 하나만 선택하여 연동**해야 하며,\
**동시에 사용해서는 안 됩니다.**

***

### 설치 방식 선택 가이드

| 프로젝트 환경            | 권장 방식          |
| ------------------ | -------------- |
| 신규 프로젝트            | XCFramework    |
| Swift 기반 프로젝트      | XCFramework    |
| 레거시 프로젝트 유지        | Static Library |
| 기존 `.a` 라이브러리 사용 중 | Static Library |
